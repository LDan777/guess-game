# 🎮 TF家族猜猜看

一个基于网页的「猜角色」小游戏，支持**单机练习**和**简单联机对战**。
灵感来源是实体时代峰峻猜角色卡牌游戏，类似 Guess Who，但主角是 TF 家族二到四代。
地址：https://ldan777.github.io/guess-game/
---

## ✨ 功能介绍

* 🎲 随机抽取你的目标角色
* 🟢 点击卡片进行「排除」
* 🔴 提交答案进行猜测
* 🧠 单机模式：比较方便，因为联机有时候不太稳定
* 🌐 联机模式：通过房间号对战（基于 PeerJS）

---

## 🕹️ 游戏玩法

1. 选择游戏模式（单机 / 创建房间 / 加入房间）
2. 点击「抽取我的角色」
3. 通过像对手提问，逐步**排除不可能的人物**
4. 切换到「提交模式」，选择你认为的答案
5. 猜对即胜 🎉

## 🌐 联机说明

* 创建房间 → 会生成房间号
* 另一方输入房间号加入
* 基于 **PeerJS (WebRTC)** 实现点对点连接

⚠️ 注意：

* 需要网络正常
* 某些网络环境（校园网等）可能连接失败
